# Clifford-back

Cliente fácil de formularios dinámicos (back end).

"Back end" RESTful de Clifford. Interactúa con el servidor de form.io y con el de jBPM

## Obtener el código

```
$ git clone https://github.com/pgirala/Clifford-back.git
$ cd Clifford-back
$ npm install --force
```

## Ejecutar la aplicación (desarrollo)

```
$ npm start
```

`http://localhost:4200/`

La aplicación se recarga cada vez que se cambia un fichero.
